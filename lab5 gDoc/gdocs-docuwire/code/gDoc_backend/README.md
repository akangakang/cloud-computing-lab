# GDoc Backend

gDoc是基于分布式文件系统的在线共享文档编辑应用。本目录为gDoc应用的后端程序。

## 项目结构

- `conf` 配置文件
- `controllers` RESTful请求处理，websocket维护
- `db` 连接数据库
- `doc` 项目文档
- `fs` 连接分布式文件系统
- `handler` 处理业务逻辑
- `lock` 锁相关
- `model` 数据结构定义，DAO
- `router` 路由配置
